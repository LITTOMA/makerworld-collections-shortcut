# MakerWorld Collection Shortcut

在 MakerWorld 导航栏添加收藏夹快捷按钮。

## 安装

### Chrome/Edge
1. 打开 `chrome://extensions/` 开启开发者模式
2. 加载 `chrome-extension` 文件夹
3. 生成图标：`pwsh tools\icon-generator.ps1`

### Firefox
1. 打开 `about:debugging#/runtime/this-firefox`
2. 临时加载 `firefox-extension/manifest.json`

### Tampermonkey
1. 安装 Tampermonkey 扩展
2. 导入 `userscript/makerworld-quickfav.user.js`
